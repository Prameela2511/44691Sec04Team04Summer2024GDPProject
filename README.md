# GDPSec04Team04
Personal Database Development Project

Project Description:
The goal of this project is to develop a personal DBMS that allows users to store, retrieve, manage, and query their data efficiently. This could be used for managing personal information like contacts, schedules, notes, or even specific data like personal finance records or a collection inventory (books, games, etc.).

Setup instructions: 
Setup Guide for Localhost and HeidiSQL with Visual Studio Code:
1. Install Visual Studio code
2. Add SQL extensions - In VS Code, go to Extensions (Ctrl+Shift+X), search for "SQL Tools" or "SQL Server (mssql)," and install.
3. Set Up Localhost Server - Ensure relevant platform is installed. In VS Code terminal, start your server. Check http://localhost:<PORT> to confirm it’s running.
4. Install Heidi SQL and connect to the database in VS Code.
5. Run and verify.
   
Steps to run the project:
1. Open Visual Studio Code and put the code in the new file.
2. save the code and then open the terminal and run the below commands for the local host server.
3. npm run develop
4. npm run db:reset
5. These are the commands that shoud be run in the terminal then we will get an local host server link which is  http://localhost:<PORT>
6. open the server  http://localhost:<PORT> in chrome and you will be redirected to the webpage of the project.
